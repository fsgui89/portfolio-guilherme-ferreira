# Guilherme Ferreira Soares | Portfolio

Personal portfolio developed to present my professional background, technical skills, projects, and positioning as a **Product Engineer** and **Full Stack Developer** focused on **JavaScript, React, Node.js, SaaS, AI, UX, and process automation**.

The portfolio was designed with English as the default language and includes language switching options for Portuguese and Italian.

## Live Preview

Portfolio:

## About the Project

This project was created to centralize my professional journey, connect my previous experience in management, education, operations, automation, and data analysis with my current transition into technology, and present my evolution through practical web development projects.

The portfolio focuses on:

* Professional positioning;
* Personal branding;
* Project presentation;
* User experience;
* Responsive design;
* Multilingual content;
* Clean structure using HTML, CSS, and JavaScript.

## Features

* English as the default language;
* Portuguese and Italian language options;
* Responsive layout;
* Fixed navigation menu;
* Hero section with professional positioning;
* About section with career background and product vision;
* Skills section with technology and product-related competencies;
* Projects section with Live Demo and GitHub links;
* Contact section with e-mail, LinkedIn, GitHub, and WhatsApp;
* Back-to-top button;
* Language preference saved with `localStorage`.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* LocalStorage
* Responsive Design
* DOM Manipulation
* Git and GitHub
* GitHub Pages
